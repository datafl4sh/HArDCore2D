Returns the diffusion matrix at the points x, y.

iTC[1]=1: Diff = Id

iTC[1]=2: Diff = [y^2+1 -xy; -xy x^2+1]

iTC[1]=3: Diff=Id if x<=1/2, Diff=[2 0; 0 1] if x>1/2. Only valid with iTC[0]=1;

◆ sol()

double TestCase::sol ( const double  x,
const double  y 
)

Returns the exact solution at the points x, y.

iTC[0]=1: u(x,y)=sin(pi x) * sin(pi y)

iTC[0]=2: u(x,y)=cos(pi x) * cos(pi y)

iTC[0]=3: u(x,y)= x

iTC[0]=3: u(x,y)= y

iTC[0]=4: u(x,y)= x^2 + y^2
